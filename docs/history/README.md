# Planning history

Original planning and decision docs from this project's first public iteration,
[`internal-ai-agent-eval-lab`](https://github.com/rosscyking1115/internal-ai-agent-eval-lab)
(now archived), kept here for provenance. They record early decisions, not the current design.

`decision_memo.md` records the decision to adopt the public "Eval Lab" identity, and
`project_direction.md` is the North Star doc that came out of it.

`safety_prevalence_module_plan.md` is the design plan for evaluating the safety classifier
under sampled or under-represented unsafe traffic. It was later implemented in this repo as
the safety-classifier review-policy study, described in the
[agent-safety intervention study](../agent_safety_intervention_study.md).
