# Planning history

Original planning and decision docs from this project's first public iteration,
[`internal-ai-agent-eval-lab`](https://github.com/rosscyking1115/internal-ai-agent-eval-lab)
(now archived), kept here for provenance. They record early decisions, not the current design.

- **`decision_memo.md`** — the decision to adopt the public "Eval Lab" identity.
- **`project_direction.md`** — the North Star doc.
- **`safety_prevalence_module_plan.md`** — the design plan for evaluating the safety
  classifier under sampled/under-represented unsafe traffic. This was later implemented in
  this repo as the safety-classifier review-policy study — see the
  [Agent-safety intervention study](../agent_safety_intervention_study.md) (the safety
  classifier + secondary review experiment).
