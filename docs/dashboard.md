# Dashboard: run locally and deploy

The Streamlit dashboard is the reviewer/evidence view of the release gate — it renders
the incident-replay verdict (ship / warn / block), the must-not violations, and the
evaluation metrics. The public, always-available surface is the
[GitHub Pages report](https://rosscyking1115.github.io/agent-release-gates/); the
dashboard is the interactive complement.

## Run locally

```powershell
uv sync
uv run python scripts/run_all_evals.py          # generate report data (also committed)
uv run streamlit run streamlit_app.py --server.port 8510
```

Open `http://localhost:8510`. `streamlit_app.py` (repo root) is the entry point; it
delegates to `app/streamlit_app.py`. The report artifacts it reads are committed under
`reports/`, so the dashboard renders without regenerating them.

## Deploy on Streamlit Cloud

1. Go to <https://share.streamlit.io> and sign in with the GitHub account that owns
   `rosscyking1115/agent-release-gates`.
2. **New app → from repo**: repository `rosscyking1115/agent-release-gates`, branch
   `main`, **main file path `streamlit_app.py`**.
3. Deploy. Because the `reports/` artifacts are committed, the app has data on first
   load — no extra build step is required.
4. **To share it publicly:** open the app's **Settings → Sharing** and set visibility
   to **Public**. Without this, the app URL redirects to a Streamlit sign-in page (the
   reason a hosted instance can look "down" when it is merely private).
5. Put the resulting URL in the README header links and in
   `scripts/build_public_site.py` (`streamlit_url`).

> The current public instance is <https://agent-release-gates.streamlit.app/>. An
> earlier deployment lived at `agent-evaluation-lab.streamlit.app` (the pre-rename
> name) and has been retired.

## Refresh the data

Report artifacts are regenerated by `scripts/run_all_evals.py` and committed to
`reports/`. To update what the deployed dashboard shows, regenerate and commit:

```powershell
uv run python scripts/run_all_evals.py
git add reports/ && git commit -m "Refresh evaluation reports" && git push
```

Streamlit Cloud redeploys on push to `main`.
